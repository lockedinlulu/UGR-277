# Week 5: AutoML & No-Code Model Training

This project builds an emergency vs routine classification model using Google Teachable Machine and compares generic vs fine-tuned NLP models using Hugging Face and n8n.

---

## Custom Model Training

- Built emergency vs routine image classifier
- Accuracy: 90%
- Precision: 100%
- Recall: 80%
- F1: 88.9%

---

## Fine-Tuned Model Comparison

Compared:
- Generic sentiment model (distilbert SST-2)
- Emotion detection model (RoBERTa emotion)
- Twitter sentiment model

Fine-tuned models performed better at identifying emergency-related emotional tone.

---

## Key Finding

Fine-tuned emotion models outperform generic sentiment models for security-related classification tasks.
